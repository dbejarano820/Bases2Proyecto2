### docker related  
To build the image and run it 
```

docker run -it --name ftv-park --rm -p 7077:7077 -v /Users/danielbejarano/Desktop/BaseDeDatos2/Proyects/Bases2Proyecto2/pyspark/examples:/src --net ftvnet --ip 10.0.0.3 pyspark bash

```

### pyspark related

spark-submit --packages org.apache.spark:spark-sql-kafka-0-10_2.12:3.0.0 read.py
